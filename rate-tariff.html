金額稅率連動（觸發 D 匯率 + 觸發 E 稅則）
2026-05-06

一、觸發 D / 觸發 E 是什麼

OP 打報單時最後幾個欄位會「自動連動」算出兩個重要數字：

  • 觸發 D = 匯率 — 報單上的外幣金額要換算成台幣，台幣金額決定稅基。匯率不是即時銀行匯率，是海關每旬公告（每月 3 次：5、15、25 日）。
  • 觸發 E = 稅率 — 每筆貨物用 11 碼商品編號（CCC 號列）查海關稅則資料，得出該繳的關稅%和簽證代號。

這兩個觸發是關稅算對的關鍵，算錯就直接漏稅或溢稅。

1.1 業務範圍

觸發應用情境影響欄位數連動深度
觸發 D 匯率幣別變更 / 日期變更 / 任一金額變更~6 個（匯率、CIF/FOB、保險費、台幣金額…）4 步：幣別→匯率→保險費→FOB/CIF→台幣稅基
觸發 E 稅則商品編號（CCC_CODE）變更~8 個（稅率、單位、簽證代號 7 組…）2 步：CCC→稅率/單位/簽證

1.2 報關員實際操作

  • 選好幣別（如 USD），系統自動帶當前旬的賣出匯率（進口用）
  • 輸入外幣金額，系統自動算台幣金額
  • 選好貿易條件（FOB / CIF / EXW…），系統按 6 種公式自動算保險費
  • 每個項次填商品編號 CCC，系統自動帶稅率 / 單位 / 簽證代號

二、觸發 D 匯率：旬別制與五大幣

海關不用即時匯率（會打不下單），改用「旬」（10 天為一旬）固定匯率。每月 3 次公告：

  • 5 日公告 → 適用上旬 1-10 日
  • 15 日公告 → 適用中旬 11-20 日
  • 25 日公告 → 適用下旬 21-月底

共 29 個幣別。實務五崧 99% 的單只用 5 大幣（USD / EUR / JPY / CNY / HKD）。

2.1 旬制規則（海關手冊明文）

旬別適用日期公告日備註
上旬每月 1-10 日當月 5 日遇假日往前推
中旬每月 11-20 日當月 15 日遇假日往前推
下旬每月 21 日 - 月底當月 25 日遇假日往前推

2.2 進口 vs 出口用不同欄位

用途欄位意義
進口報關賣出價 sellValue銀行賣外幣給你的價（較高）
出口報關買入價 buyValue銀行向你買外幣的價（較低）

資料來源：關稅法第 39 條 + 關務署 GC331「進口貨物完稅價格適用之外幣匯率」公告

六、保險費 6 種公式（觸發 D 連動下游）

選了「貿易條件」（FOB / CIF / EXW…）後，系統會用 6 種不同的數學公式自動算保險費，連動到台幣稅基（CIF）。

6.1 6 種貿易條件公式

貿易條件意義保險費公式
FOB / FAS / EXW賣方只負責貨到港 / 船邊 / 工廠保險 = 比率 × (金額 + 運費 + 加減項)
CIF / CIP賣方含保險 + 運費從 CIF 反推 FOB 後計算
C&I（成本+保險）賣方含保險不含運費特殊公式
C&F / CFR賣方含運費不含保險保險 = (金額 + 加減) × 比率
國內運送（運輸方式 5/99）不過國境不算保險

七、觸發 E 稅則：海關三欄稅率制

同一個商品對不同國家的關稅率不一樣。海關設計三欄稅率：

  • 第一欄 = 一般稅率（多數國家）
  • 第二欄 = FTA 優惠稅率（巴拿馬 / 瓜地馬拉 / 尼加拉瓜 / 薩爾瓦多 / 宏都拉斯 / 紐西蘭 / 新加坡 7 個自由貿易協定國）
  • 第三欄 = 高稅率（懲罰）

另外政府不定期會公告「機動稅率」臨時降低某類商品稅率（例如生鮮牛肉），蓋過第一欄。

8.2 半導體積體電路特別案例

九、五崧現行 prod 稅則維護機制

海關沒有提供公開 API（不像匯率），只能 PDF 下載。LRP 系統有內建的「稅則維護介面」+「F5 批次匯入」按鈕，由 IT 拿海關 PDF 轉文字檔後一次匯入。

9.1 現行流程（半人工半程式）

  1. 海關公告新版稅則 PDF（每年 1/1 + 7/1 改版 + 不定期機動稅率）
  2. IT 將 PDF 轉成文字檔
  3. 登入 LRP「稅則維護介面」（PLB020）
  4. 按「F5:Import」選檔案
  5. 系統自動批次匯入 7 萬多筆稅則 + 自動標記資料來源為 Batch_IMP

十、簽證代號（IM_PMT / EX_PMT）
